🎯 HOW TO USE
════════════════════════════════════════════════════════════════════════════════

For Admins:
  1. Navigate to Admin Dashboard → Orders → View All Orders
  2. Click "+ Create New Order" button
  3. Select customer type (Guest or Registered User)
  4. Fill in customer information
  5. Search and add products
  6. Select variants if available
  7. Set quantities
  8. Select shipping method
  9. Choose payment method & status
  10. Apply discount if needed
  11. Add admin notes (optional)
  12. Review order summary
  13. Click "Create Order"
  14. Order created with orderSource="admin"

Viewing Created Orders:
  • Look for the "Order Source" column in the order list
  • Admin orders show "Admin" badge
  • Web orders show "Web" badge

🔧 API ENDPOINT
════════════════════════════════════════════════════════════════════════════════

POST /api/orders/admin/create

Authorization: Bearer {adminToken}
Permission Required: edit_orders

Request Body:
{
  "userId": "user_id_or_null",
  "items": [
    {
      "productId": "product_id",
      "variantId": "variant_id_or_null",
      "quantity": 2,
      "price": 1500
    }
  ],
  "shippingInfo": {
    "fullName": "Name",
    "mobileNo": "01234567890",
    "email": "email@example.com",
    "address": "Address"
  },
  "shippingId": "shipping_id",
  "paymentMethod": "cash_on_delivery",
  "paymentStatus": "unpaid",
  "orderSource": "admin"
}

Response:
{
  "success": true,
  "message": "Admin order created successfully",
  "order": { ... full order object with orderSource="admin" ... }
}

❓ COMMON QUESTIONS
════════════════════════════════════════════════════════════════════════════════

Q: What's the difference between web and admin orders?
A: Both are identical orders, just marked with different orderSource values
   for tracking/reporting purposes.

Q: Can I create orders for guests?
A: Yes! Toggle to "Guest Checkout" and enter customer details manually.

Q: What if a product doesn't have variants?
A: The component handles it gracefully - it uses the product's default price
   and skips the variant selector.

Q: Can I edit orders after creating them?
A: Yes, use the existing edit functionality (same as web orders).

Q: Do admin orders get stock deducted?
A: Yes, stock is deducted immediately just like web orders.

Q: What permissions does the admin need?
A: The admin must have the "edit_orders" permission.

See TROUBLESHOOTING_ADMIN_ORDERS.md for more FAQs.
